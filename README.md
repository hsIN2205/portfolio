# 실행하는법

1. git clone https://github.com/hsIN2205/portfolio.git
2. cd 다운받은폴더
3. npm install
4. npm start

# 수정한거 올리기전에

1. git add . (작업한거 추가하고)
2. git commit -m "작업내용"
3. git pull
4. 충돌 일어난거 있으면 수정
5. 제대로 실행되는지 확인
6. git push
  - 만약에 push 안되면 git push -u origin master
